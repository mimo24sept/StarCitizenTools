Place your custom sidebar background images here.

Expected filenames:
- `crafting.png` / `crafting.jpg` / `crafting.jpeg` / `crafting.webp` / `crafting.svg`
- `trade.png` / `trade.jpg` / `trade.jpeg` / `trade.webp` / `trade.svg`
- `loadouts.png` / `loadouts.jpg` / `loadouts.jpeg` / `loadouts.webp` / `loadouts.svg`
  or `loadout.png` / `loadout.jpg` / `loadout.jpeg` / `loadout.webp` / `loadout.svg`
- `wikelo.png` / `wikelo.jpg` / `wikelo.jpeg` / `wikelo.webp` / `wikelo.svg`

Priority order used by the app:
1. `.png`
2. `.jpg`
3. `.jpeg`
4. `.webp`
5. `.svg`

So if you add `crafting.png`, it will automatically be used instead of `crafting.svg`.
